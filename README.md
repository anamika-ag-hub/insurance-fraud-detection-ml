# insurance-fraud-detection-ml

## Project Overview
This project focuses on detecting fraudulent insurance claims using Machine Learning algorithms. The objective is to classify whether an insurance claim is fraudulent or non-fraudulent based on various customer and claim-related features.

---

## Dataset
The dataset used in this project contains insurance claim details such as:
- Customer information
- Incident details
- Claim amount
- Vehicle information
- Fraud report status

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib

---

## Machine Learning Models Used
1. Logistic Regression
2. Decision Tree Classifier
3. Random Forest Classifier
4. Naive Bayes Classifier

---

## Data Preprocessing
The following preprocessing techniques were applied:
- Handling missing values
- Replacing '?' values
- Removing unnecessary columns
- Label Encoding
- Outlier Detection and Removal using IQR Method

---

## Exploratory Data Analysis (EDA)
EDA was performed using:
- Count Plots
- Histograms
- Correlation Heatmap
- Boxplot for Outlier Detection

---

## Model Evaluation
The models were evaluated using:
- Accuracy Score
- Classification Report
- Confusion Matrix

---

## Project Type
- Supervised Machine Learning
- Binary Classification

---

## Files Included
- `insurance_fraud_detection.ipynb`
- `fraud_insurance_claims.csv`
- `insurance_fraud_model.pkl`

---

## Conclusion
Different machine learning models were trained and evaluated for fraud detection. Decision Tree Classifier achieved the best accuracy among the implemented models for this dataset.
